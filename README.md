# Idea

## Expense & Budget Tracker

## Core Idea
Track personal expenses with categories, visualize trends, and optionally share budgets with friends/family.

## Stack
- **Backend:** Node.js + TypeScript (Express)
- **Frontend:** React + CSS
- **Database:** MySQL

## Core MVP Features (Fast to Build)
- User login & registration
- Add/edit/delete expense items
- Categorize expenses
- Simple dashboard showing total spending and per-category
